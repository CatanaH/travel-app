# travel-app

Travel Planner/Tracker/Budgeter

The goal of the program is to track travel plans: to stay organized and track how much money spent

the user will be able to:
  1. Create a trip
  2. add costs to the trip
  3. delete cost items
  4. edit cost items
  5. View a trips total plans
  6. View daily items
costs for trip include:
  • Transportation
  • lodging
  • events
  • meals
  • merchandise
  • fees

My goal with this program:

make a working tracker that saves data for later viewing, track a trips running total, when provided the date can return the next schedualed item on the planner
